# 清烟之旅 - 网页版

戒烟精灵养成应用，每日打卡守护健康。

## 功能特点

- 🌱 **精灵养成** - 戒烟天数增加，精灵会进化成长
- ✅ **每日打卡** - 坚持打卡获得能量，保护你的精灵
- 🏆 **成就系统** - 达成里程碑解锁勋章
- 📊 **戒烟统计** - 记录天数、省钱金额、少吸烟数量
- 📱 **PWA支持** - 可安装到手机桌面，如原生App体验

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 部署到 Vercel

1. 将项目推送到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成

## 数据存储

数据保存在浏览器 localStorage，无需登录注册，适合个人使用。
